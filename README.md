\# FitMind Bot



Telegram-бот для фітнес трекінгу та персональної статистики.



\## Функціонал



\- Реєстрація користувача через анкету

\- Розрахунок ІМТ та BMR

\- Профіль користувача

\- Система цілей

\- Випадкові тренування

\- Логування виконаних вправ

\- Статистика активності за 7 / 30 днів

\- Меню кнопками



\## Технології



\- Python 3.11+

\- aiogram 3.x

\- SQLite

\- aiosqlite

\- python-dotenv



\## Структура проєкту



\- handlers/ — логіка команд

\- core/ — база даних та конфігурація

\- keyboards/ — клавіатури

\- services/ — обчислення



\## Локальний запуск



```bash

pip install -r requirements.txt

python main.py
