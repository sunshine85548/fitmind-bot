# FitMind Bot

Telegram-бот для фітнес трекінгу та персональної статистики.

## Функціонал

- Реєстрація користувача через анкету
- Розрахунок ІМТ та BMR
- Профіль користувача
- Система цілей
- Випадкові тренування
- Логування виконаних вправ
- Статистика активності за останні 7 / 30 днів
- Меню кнопками

## Технології

- Python 3.11+
- aiogram 3.x
- SQLite
- aiosqlite
- python-dotenv
- pytest
- GitHub Actions

## Структура проєкту

- `handlers/` — логіка команд та обробників
- `core/` — база даних і конфігурація
- `keyboards/` — клавіатури Telegram
- `services/` — математичні обчислення
- `tests/` — unit тести

## Запуск проєкту

```bash
pip install -r requirements.txt
python main.py
